# 特色民宿在线预订平台
# homestay



## 项目简介
1.本项目是基于SpringBoot的特色民宿在线预订平台，实现特色民宿的集中化、标准化管理。
2.主要用于企业后台管理
3.核心功能介绍：该平台涵盖用户登录、房源搜索、订单管理以及特色服务管理等诸多功能模块，并创新性地引入第三方
管理机构角色，旨在优化用户体验并提升管理效率。在开发过程中，运用 Spring Boot 框
架以简化后端开发流程，同时结合 VUE 技术精心构建前端界面，从而确保整个系统的稳定
性和可扩展性。该平台不仅提高了民宿的管理效率，也为房客提供了更便捷、个性化的服
务体验。通过性能测试，验证了系统在高并发场景下的良好性能。

## 技术栈
### 后端技术
- 核心语言：Java JDK1.8 / Python3.9
- 框架：SpringBoot、MyBatis、Spring MVC
- 数据库：MySQL 8.0、Redis
- 中间件：Maven、Tomcat

### 前端技术（无前端可删掉）
- 基础：HTML5 + CSS3 + JavaScript
- 框架：Vue2 / Element UI

### 开发&工具
- IDE：IDEA / VS Code
- 版本控制：Git / GitHub

## 项目结构
- controller：接口控制层
- service：业务逻辑层
- mapper：数据库操作层
- entity：实体类
- resources：配置文件资源
